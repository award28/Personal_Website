Studi Buddi

Studi Buddi was built at YHack 2015. We made this website due to a problem we felt was prevalent at our school; finding the right person to study with. While studying with your friends is always a great option, most of the time they aren't studying the same class, let alone the same major. Studi Buddi looks to close this studying gap by using geo-location services to bring students studying the same subject together. User generated studi-hubs give off location and the current subject the user is studying. When the user leaves their studying location, they are removed from the list of studi-hubs. This website was built with: Facebook API, google maps API, node.js, mongodb, postman, jquery, javascript, html, and css.

Sealify

Sealify.me was created by my team and me at BrickHack2, during the spring 2016 semester. We decided to encrypt email using pgp private and public key encryption. Rather than decrypting on the server like many mail services that are encrypted, our service encrypts the email client-to-client. Our service allows users to relax knowing their data is safe and only seen by them and the receiver. This project was built with Node.js, JavaScript, AngularJS, ejs, Bootstrap , Keybase, Google­-compute­-engine, MongoDB, Redis, and Letsencrypt. We took home “Best use of Encryption” and “Best Reverse Engineering Hack” for this hack.

  • Hackathon: BrickHack2
  • Date: March 2016

Fifty Twenty Thirty

Considering many individuals struggle with personal finance, I saw the need to develop a tool which would help solve this common problem. Following Elizabeth Warren's 50-20-30 rule for money management, I applied this technique to automatically split up users paycheck accordingly. This rule states that 50% should go towards short term necessities, 20% towards long term retirement goals, and 30% for entertainment. This web app was built with express.js, angular.js, node.js, morris.js, ejs, and Capital One's API.

  • Hackathon: MLH Prime
  • Date: August 2016

Banking Simpatico

Created at BigRed//Hacks 2015, Banking Simpatico is designed to be a simple dashboard for banking. We used the Capital One API (Nessie) To simulate user data, and incorporated the Priceline API to gather information on possible travel destinations. Designed to be a user-friendly dashboard, Banking Simpatico allows you to oversee your family's finances, and explore reward opportunities such as cashback and free miles. This website was built with Python using Flask, and took advantage of many resources including: Capital One API, Priceline API, d3.js, postman, google-material-design, heroku, pop.co, jquery, javascript, html, and css.

Food Bites

While attending a class with my friend John Reynolds, we were discussing the hassle one must go through to look up a recipe. We decided there is a better way to handle this, and as such we created Food Bites. This website looks up recipes, and if you decide you like the recipe, you click a button to add the ingredients to a shopping list. This web app was built with Node.js, JavaScript, AngularJS, ejs, Bootstrap, Python, Beautifulsoup, Flask, Requests, and Let’s Encrypt.
